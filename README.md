# YouTube UI Clone

A YouTube homepage clone built with Angular and Tailwind CSS.

## Versions

- **Angular**: 18.0.0
- **Node.js**: v18 or higher
- **TypeScript**: 5.4.2
- **Tailwind CSS**: 3.4.0

## How to Run

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm start
   ```

3. **Open your browser:**
   ```
   http://localhost:4200
   ```

## Build for Production

```bash
npm run build
```

## Available Scripts

- `npm start` - Start development server
- `npm run build` - Build for production
- `npm test` - Run tests
